Privacy Policy

Last updated: March 17, 2026

Etsy Tag Studio ("the Extension") is a Chrome browser extension that helps Etsy sellers manage SEO tags. This policy describes what data the Extension handles and how.

Data We Do Not Collect

The Extension has no backend server and does not transmit any data to the developer. We do not collect, store, or have access to any personal information.

Data Stored Locally on Your Device

The Extension stores the following data in your browser's local storage (chrome.storage.local), accessible only to the Extension on your device:

  • OpenAI API key — entered by you in Settings; used only to call the OpenAI API on your behalf.
  • License key and activation record — your Lemon Squeezy license key and activation metadata, used to verify your license.
  • Project data — tag pools, projects, and settings you create within the Extension.
  • eHunt session token — a session token captured from ehunt.ai, used only to query the eHunt tag API on your behalf.

Third-Party Services

The Extension communicates with the following external services directly from your browser. No data passes through the developer's servers.

  • OpenAI API (Privacy Policy) — When you use AI features, listing images or text you provide are sent to the OpenAI API using your own API key. OpenAI's privacy policy governs this data.
  • eHunt API (ehunt.ai) — Etsy listing IDs and search queries are sent to the eHunt tag data API to retrieve tag statistics.
  • Lemon Squeezy License API (Privacy Policy) — Your license key is sent to Lemon Squeezy to activate and validate your license. Lemon Squeezy's privacy policy governs this data.
  • Etsy (Privacy Policy) — The Extension reads publicly visible tag and listing data from Etsy pages you visit. No data is written to Etsy on your behalf.

Permissions

The Extension requests only the permissions necessary for its features:

  • storage — to save your projects and settings locally.
  • tabs — to open the dashboard and navigate Etsy search pages during tag collection.
  • Host access to etsy.com, ehunt.ai, api.openai.com, api.lemonsqueezy.com — to read listing data and communicate with the third-party APIs described above.

Children's Privacy

The Extension is not directed at children under 13 and does not knowingly collect information from children.

Changes to This Policy

If this policy is updated, the "Last updated" date above will change. Continued use of the Extension after changes constitutes acceptance of the updated policy.

Contact

Questions about this privacy policy can be sent to the developer via the Chrome Web Store listing page.
